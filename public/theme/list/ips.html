Whitelisting DownNotifier.com

If your website has IP access restrictions or rate limiting, you may need to whitelist DownNotifier.com. You can do this by IP address, or by user agent.

Whitelist by IP address

Our IP address list is only available for premium and professional monitoring members. Free accounts use a more flexible number of servers, so unfortunately we can't provide IP addresses for free accounts. Please login first to show the list.

Whitelist by user agent

DownNotifier uses the user agent string "downnotifier.com monitoring". When you whitelist this user agent all DownNotifier monitoring actions will pass.
